# Typescript; Type & Interface

## Type vs Interface

* TypeScript 기본 타입
    * boolean
    * number
    * string
    * array
    * tuple
    * enum
    * unknown
    * any
    * void
    * null
    * undefined
    * never
    * object


* Type vs Interface
    * Interface
        * a way to describe data shape
    * Type
        * definition of a type of data, for example, a union, primitive, intersection, tuple, or any other type

* Interface만 가능한 것
    * Declaration Merging
        * extends
        * implements

* Type만 가능한 것
    * Intersection
    * Union
    * Tuple

## 어떤 걸 써야할지

* Interface는 새로운 객체나 객체의 메소드를 정의할 때 유용함
* Type은 함수를 정의할 때 유용함 (또는 복잡한 타입)
